# Roster Bot

Roster Bot is a Discord RPG bot created as a learning project. Its main purpose is to help me learn Discord bot development, database integration, and asynchronous programming in Python.

The bot allows users to perform CRUD (Create, Read, Update, and Delete) operations on their own RPG character data through Discord commands and interactions.

# Technologies
Python
Pycord (Discord API wrapper)
aiosqlite
python-dotenv
Current Status

The project is still under development. Some features are not yet ready for deployment, so they remain in the development branch of the repository.

# Future Plans

As I continue learning, I plan to add new features and improve the bot's functionality, code structure, and overall user experience.
